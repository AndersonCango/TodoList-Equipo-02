Metodologías Ágiles - Ingeniería en Software

Esta aplicación ha sido desarrollada como parte del proyecto académico de la asignatura de Metodologías Ágiles, utilizando el framework Spring Boot como núcleo tecnológico y aplicando metodologías ágiles con uso de herramientas como Trello & GitHub Projects para su organización y desarrollo iterativo.

El sistema permite gestionar tareas de forma eficiente, facilitando la organización personal.. Su diseño modular, interfaz intuitiva y escalabilidad hacen de esta herramienta una solución óptima para entornos académicos y personales.
